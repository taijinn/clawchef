# ClawChef
## Your Graphical Interface for OpenClaw

ClawChef is an independent, open-source project designed to provide a fast, intuitive, and easy-to-use graphical setup interface for the [OpenClaw](https://github.com/openclaw/openclaw) AI agent orchestrator.

### ⚠️ IMPORTANT: Read the Disclaimer

### Features
* **Zero-CLI Setup:** Bypass the terminal and instantly manage your OpenClaw configurations.
* **AI Provider Management:** Easily toggle your preferred AI backends (OpenAI, Anthropic, Google Gemini).
* **Channel Dashboard:** Visually manage your bot integrations across WhatsApp, Telegram, and Lark (Feishu).
* **Built in Electron:** A cross-platform desktop manager constructed strictly with Vanilla JavaScript and Vite.
